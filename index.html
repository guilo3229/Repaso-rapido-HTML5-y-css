Etiquetas semanticas estructurales

div es una etiqueta contenedora que no tiene semántica
El Header es la cabezera de una web o un sitio
main- Define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer - Pie de página de un sitio web o de una seccion
nav representa una navegacion ya sea botonera o/y para la pagina web
article -representa una sección autocontenido (que por si sola se explica a ella misma)
aside- Representa contenido complementario o secundario(publicidad)
Section - Representa una seccion de contenido generico
address - Representa una dirección FÍSICA

Etiquetas de bloque VS etiquetas de Línea

La etiqueta de bloque por referencia es la div, una etiqueta de bloque es aquella que genera slato de linea con sus etiquetas hermanas
La etiqueta de linea por referencia es la span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido. Hola soy otra span 🖢
